## Overview
Focusing on custom field support and dynamic schema rendering to unlock high-utility workflows.

## Capabilities
- **Direct Custom Field Population:** Populate tenant-specific custom fields on Intakes and People via Zapier.
- **Dynamic Schema Rendering:** Key engineering win allowing Zapier to "understand" and display a customer's specific Casebook schema automatically in the Zap builder.
- **Extended Lookup/Write:** Support for "Search Cases" and "Create a Note" actions.

## Key Benefits
- **No Developer Required:** Enables admin-level users to build complex integrations without API expertise.
- **Bridge to Intake:** Link any number of people to a new intake report automatically.
