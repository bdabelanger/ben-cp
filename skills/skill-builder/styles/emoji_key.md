# Emoji Key: Standardized Health Indicators

> [!NOTE]
> 🎨 **PURPOSE:** This file defines the standardized emoji usage across all Skills to ensure visual consistency in final reports.

---

## ✅ Success & Completion
*   **✅ Done/Achieved:** Used when a Key Result (KR) has met or exceeded its target. *Trigger: Current Value $\ge$ Target Value.*
*   **👏 Best Performer:** Reserved for the highest-performing entity in a given metric category.

## ⚠️ Caution & Risk
*   **⚠️ At Risk/Needs Attention:** Used when performance is trending poorly or data is incomplete but not yet failed. *Trigger: Current Value is approaching Target Value (e.g., >80%) OR required data fields are missing.*
*   **🎯 On Track/Action Required:** Used for items that require monitoring or have no defined target yet.

## ❌ Failure & Blockers
*   **❌ Missed/Failed:** Used when a KR has definitively failed to meet its target. *Trigger: Current Value $<$ Target Value.*

## 👀 Data Quality / Missing Information
*   **👀 Needs Attention (Data):** Used specifically when required data points are absent or incomplete, regardless of performance.
    *   *Example:* No current value logged for a KR.

---

## 🔗 Usage Note

When documenting a Skill's `styling.md`, reference this file to ensure your chosen indicators align with the organizational standard.
