# Changelog: Product Intelligence

All notable changes to the Product Intelligence domain will be documented in this file.

## [1.0.0] — 2026-04-13

**Initial Codification:**
- Created `intelligence/product/projects/` directory.
- Extracted contextual metadata for 35 active projects from `asana_active.json`.
- Generated standardized project status mappings for the entire Platform portfolio.
- Established master and projects directory indexing.
- **Refactoring:** Refined project filenames to human-readable kebab-case (e.g., `nylas-upgrade-ux-improvements-(1208822133040792).md`).
- **Hierarchy:** Implemented year/quarter hierarchical grouping for all projects, highlighting the active 2026 Q2 portfolio.
- **Partitioning:** Segregated legacy 2025 context into `archive/` and unscheduled initiatives into `backlog/`.
- **Enrichment:** Cross-referenced Q2 Product Shareout requirements to populate active project files with specific "Slide-level" documentation needs.
